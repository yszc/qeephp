# 开发环境 #

开发环境为 Windows。单元测试分别在 Windows 和 CentOS5 中运行。

# 测试环境 #

发布前，将在 Windows + Apache、Windows + IIS、CentOS5(RHEL5)、Ubuntu 中进行单元测试和整合测试。

## 测试工具 ##

  * PHPUnit + xdebug
  * ApacheBench

## 测试的 PHP 版本 ##

  * PHP 5.1.x
  * PHP 5.2.x

## 测试的数据库 ##

  * MySQL 4.1/5.0
  * PostgreSQL 8
  * Oracle 10g Express
  * MS SQL Server 2005 Express
