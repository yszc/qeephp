# 助手、行为插件、用户界面控件的命名方式 #

```

一般助手、控制器助手、视图助手
类名：Helper_Array、Helper_Uploader、Helper_Html
对应文件：helper/array.php、helper/uploader.php、helper/html.php

行为插件
类名：Model_Behavior_Tree
对应文件：model/behavior/tree.php

用户界面控件
类名：Control_Pagination
对应文件：control/pagination.php

用户界面控件模板：
对应文件：view/_controls/pagination_control.php

用户界面元素
对应文件：view/_elements/header_element.php

```
